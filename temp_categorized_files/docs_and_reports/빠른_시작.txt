-------------------------------------------------------------------------------
⚠️ 문제 해결
-------------------------------------------------------------------------------

Q: Chrome이 열리지 않아요
A: Chrome 브라우저를 최신 버전으로 업데이트하세요

Q: "모듈을 찾을 수 없음" 오류
A: 명령 프롬프트에서 실행:
   python -m pip install selenium pandas openpyxl webdriver-manager

Q: 데이터가 수집되지 않아요
A: 인터넷 연결을 확인하고 메트라이프 웹사이트 접속 가능한지 확인
