## 🔍 데이터 종류별 상세 정보

### 1. 시계열 데이터 (주식, 지수, ETF 등)
**제공 컬럼**: `Open`, `High`, `Low`, `Close`, `Volume`, `Change`

### 2. 종목 리스트 데이터
**제공 컬럼**: `Code`, `Name`, `Market`, `Close`, `Volume`, `Amount`, `Marcap`, `Stocks` 등

### 3. 경제 지표 데이터 (FRED)
**제공 컬럼**: 지표별로 다름 (주로 단일 값)

## 🚀 빠른 시작 코드

```python
import FinanceDataReader as fdr

# 1. 한국 주식 전체 리스트
stocks = fdr.StockListing('KRX')

# 2. 특정 종목 데이터 (삼성전자)
samsung = fdr.DataReader('005930', '2020-01-01')

# 3. 미국 주식 데이터 (Apple)
apple = fdr.DataReader('AAPL', '2020-01-01')

# 4. 지수 데이터 (KOSPI)
kospi = fdr.DataReader('KS11', '2020-01-01')

# 5. 환율 데이터
usd_krw = fdr.DataReader('USD/KRW', '2020-01-01')

# 6. 암호화폐 데이터
btc = fdr.DataReader('BTC/USD', '2020-01-01')

# 7. ETF 데이터
spy = fdr.DataReader('SPY', '2020-01-01')

# 8. 경제 지표 (FRED)
gdp = fdr.DataReader('FRED:GDP', '2000-01-01')
```
